# MAC Agents
This is a script orchestrator that run on your mac that allows you schedule scripts that need to be run on your mac.

# Features
- a Status bar icon that shows status of all the scripts that are schedules and a countdown on when it will run next
- notification when the script runs
- status of the script - success / fail
- any data that the script returns like weather , battery charge percentage
- a simple UI to add new scripts
- it should support vens if is is a pythin script
